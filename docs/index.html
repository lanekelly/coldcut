Coldcut

The installer for AI text adventure games

Download for Windows

Instructions:

  1. Click Download for Windows button above.
  2. File coldcut-win-x64.zip will start downloading.
  3. Unzip and open folder.
  4. Double click coldcut.exe to start.
  5. Check out the included README.txt for more info.

Available games:

  • AIDungeon
  • thadunge2
  • CloverEdition
  • Storybro
  • ZenDungeon

Features:

  • Automates the full installation process.
  • Smart model sharing saves your disk space.
  • 100% self-contained. Doesn't litter your PC with developer tools.
